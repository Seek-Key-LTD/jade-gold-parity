# jade-gold-parity

玉金等价：牧月记三部曲

## 项目简介

本项目收录了"牧月记"三部曲学术论文，探讨从史前文明到现代社会的演化规律，涵盖技术、地理、人文三个维度。

```

├── mulanji/

│   └── III.md        # 牧月记（下）人卷
├── README.md
└── LICENSE
```

## 论文内容

### 牧月记（上）技术篇

探讨黑洞信息悖论、引力熵、克尔黑洞对角化等前沿物理问题，挑战传统星云假说，提出三元合并模型。

### 牧月记（下）地卷

提出"地球液压传动系统"理论，批判传统水循环/大陆漂移学说，分析文明演化规律。

### 牧月记（下）人卷

分析海市蜃楼现象、紫色染料文化"出口-再进口"模式、魏晋玄学思潮，批判西方线性分类法。

## 核心观点

- 圣火文明是跨地域共享成果
- 中亚资源与中原王权的结合路径
- 火纹演变与铜料来源同步
- 资源+王权的双重认证机制
